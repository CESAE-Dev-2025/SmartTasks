## Sobre o Projecto

<img src="images/screenshot_cover.png" style="width: 520px" alt="SmartTask Screenshot">

Esse projeto apresenta uma ferramenta de gestão inteligente (e simples) de tarefas. É composto por uma única página e faz uso dos recurso locais do seu browser.

Aqui podes:

1. Adicionar novas tarefas
2. Editar tarefas existentes
3. Marcar tarefas como concluídas
4. Eliminar tarefas
5. Filtrar tarefas

A aplicação possui 2 temas:
- Tema claro (Light Mode)
- Tema escuro (Dark Mode)

Por fim, a aplicação possui 2 idiomas disponíveis:
- Inglês
- Português

<!-- GETTING STARTED -->
## Como utilizar

Para utilizar este projeto, basta clonar o repositório.

### Pré-requisitos

É preciso ter o Git instalado.

### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/CESAE-Dev-2025/SmartTask.git
   ```
2. Modifique a URL do git remote para evitar `pushes` acidentais para o projeto base
   ```sh
   git remote set-url origin github_username/repo_name
   git remote -v # confirma as alterações
   ```
